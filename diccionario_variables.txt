                   DICCIONARIO DE VARIABLES
=============================================================

Descripción general:
-------------------------------------------------------------
El dataset registra publicaciones de prendas en una plataforma
digital. Cada fila representa una prenda publicada, con datos
relacionados con su precio, fecha de publicación y atributos
asociados.

El objetivo del análisis es estudiar la evolución temporal de las
publicaciones y descubrir patrones mediante regresión, clasificación
y agrupamiento aplicados en el contexto de Business Intelligence.
-------------------------------------------------------------

VARIABLES PRINCIPALES
----------------------------------------------------------------------------
| Nombre         | Tipo de dato | Descripción breve                        |
|----------------|--------------|------------------------------------------|
| timeCreated    | datetime     | Fecha y hora de creación de la publicación |
| retailPrice    | float        | Precio base de venta                      |
| pricePerWeek   | float        | Precio semanal de uso                     |
| pricePerMonth  | float        | Precio mensual de uso                     |
| brand          | string       | Marca o diseñador de la prenda            |
| category       | string       | Tipo o categoría de prenda                |
| color          | string       | Color principal del artículo              |
| condition      | string       | Estado del producto (nuevo o usado)       |
| size           | string       | Talla de la prenda                        |
| reciente       | int (0/1)    | 1 = Publicación posterior al año 2022     |
| mes            | datetime     | Mes de la publicación                     |
| publicaciones  | int          | Conteo mensual de publicaciones           |
| cluster        | int (0–2)    | Grupo asignado por el modelo K-Means      |
----------------------------------------------------------------------------

VARIABLES DERIVADAS
-------------------------------------------------------------
| Variable | Tipo  | Propósito principal                           |
|-----------|-------|-----------------------------------------------|
| t         | int   | Índice temporal (meses transcurridos)         |
| reciente  | int   | Clasifica publicaciones antes/después de 2022 |
| cluster   | int   | Agrupa meses según actividad (baja/media/alta)|
-------------------------------------------------------------

TRANSFORMACIONES Y LIMPIEZA
-------------------------------------------------------------
• Conversión de fechas con `pd.to_datetime(errors='coerce')`
• Eliminación de duplicados con `drop_duplicates()`
• Conversión de precios a valores numéricos con `to_numeric(errors='coerce')`
• Sustitución de nulos en precios por 0
• Agrupación mensual mediante `groupby('mes').size()`
• Escalado de variables numéricas con `StandardScaler()` para clustering
• Creación del dataset limpio final `df_limpio`

-------------------------------------------------------------
CALIDAD Y USO
-------------------------------------------------------------
• Dataset sin duplicados ni valores inválidos.
• Variables numéricas estandarizadas para modelamiento.
• Cobertura temporal: publicaciones desde aproximadamente 2014 hasta 2024.
• Dataset utilizado para análisis temporal, predicción y segmentación.
